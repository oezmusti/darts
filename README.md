<p> Der Darts Tracer ist eine Web App die die Züge innerhalb eines Dart-Spiels für bis zu voer personen Tracken kann.</p>
<br>
<p>Hierbei kann ein Spiel gestartet und gespielt werden. Die Vergangenen SPiele werden mit Datum, den Spielern und dem Gewinner in einer eigenen Historie angezeigt. DDie Spieler können ihren fortschritt tracken und gemeinsam das Dart-Spiel verfolgen</p>
<br>
Hierbei werden die Daten lokal im local storage gespeichert und werden nicht mit einer Datenbank verknüpft
